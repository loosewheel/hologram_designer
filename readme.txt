Hologram
Design hologram tables for lwcomponents:hologram minetest mod.

New - Clear the whole design to blank.
Clear - Clear the current layer to blank.
Open - Open a saved design (binary).
Save - Save the current design to a file (binary).
Copy - Copy the current design to the clipboard as a lua table.
About - This dialog.

Layout
Levels are from bottom to top. View is looking down on the design with
the hologram block in the center facing down.

Select a color and left click or drag on the cells to color them. The
light brown color is empty (no color).
